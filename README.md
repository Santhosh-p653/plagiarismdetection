# Text Similarity Checker

A simple NLP application built using spaCy and Gradio that compares two documents and calculates semantic similarity.

## Features

- Semantic document comparison
- Similarity score calculation
- Interactive Gradio web interface
- NLP-powered text analysis

## Tech Stack

- Python
- spaCy
- Gradio

## Installation

pip install -r requirements.txt
python -m spacy download en_core_web_md

## Run

python app.py
